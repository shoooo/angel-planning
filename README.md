# 株式会社エンジェルプランニング コーポレートサイト

## デモ概要

自動車部品販売事業（B2B）のコーポレートサイト提案用デモ。
ブラウザで `index.html` を直接開くだけで動作する静的サイト。

## ファイル構成

```
angel-planning/
├── index.html        # トップページ
├── about.html        # 会社概要・事業案内
├── products.html     # 製品紹介（ダミーデータ9件）
├── contact.html      # お問い合わせフォーム（バリデーションのみ）
├── robots.txt        # Disallow: / （全クローラー排除）
├── assets/
│   ├── css/style.css # 全ページ共通スタイル
│   └── js/main.js   # ハンバーガーメニュー・スクロールアニメ・フォームバリデーション
└── README.md
```

## 確認方法

1. `index.html` をブラウザで開く（Chrome / Edge / Safari 対応）
2. ナビリンクで全4ページの遷移を確認
3. DevTools でモバイル表示確認（iPhone SE / iPhone 14 等）
4. HTMLソースで `<meta name="robots" content="noindex,nofollow">` を確認
5. `robots.txt` の内容確認

### 製品管理機能（本番フェーズで別途構築）

クライアント自身が製品ページを更新できる、極限まで簡素化した管理画面を構築する。

**推奨技術スタック：**
- バックエンド：Node.js + Express
- データベース：SQLite（シンプル・追加コストなし）
- フロントエンド：HTML/CSS（既存デザインと統一）
- 管理画面：パスワード認証 + 製品CRUD（追加・編集・削除）

**データフロー：**
```
管理画面（フォーム入力）
  → SQLite保存 or JSON生成
  → 製品一覧ページが読み込み表示
```

**管理画面 機能スコープ（最小構成）：**
- ログイン（固定ID/パスワード）
- 製品一覧表示
- 製品追加（型番・製品名・カテゴリ・説明・画像アップロード）
- 製品編集・削除

### 検索エンジン対策（現状維持）

- `robots.txt` は `Disallow: /` を維持
- 各ページ `<meta name="robots" content="noindex,nofollow">` を維持
- DM・QRコード経由のアクセスのみを想定
